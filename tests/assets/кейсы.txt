1. (демка: простая карта)
+ открыть карту на 13 левеле (ОР: тайлы ожидаемой местности)
+ кликнуть по району (ОР: подсветка района, балун с названием местности)
+ призумиться на 17 левел (ОР: подсветка на месте, зум на 17 уровне)
+ кликнуть по улице (ОР: подсветка улицы, балун)
+ кликнуть по зданию (ОР: балун с информацией о здании по которому был клик)
+ показать список организаций в здании (ОР: список организаци в здании)
+ проскролить список организаций (ОР: список проскролен)
+ клинуть по соседнему зданию (ОР: предыдущий балун схлопнулся, открылся новый)
+ раскрыть список организаций
+ закрыть балун нажатием на крестик (ОР: балун схлопнулся, на карте только тайлы)

2. (демка: карта с тремя статическими дом-маркерами у которых есть хинты, добавленные разными способами: L.DG.label, bindLabel, label)
- открыть карту со статическим маркером и хинтом (ОР: на карте есть маркер)
- перейти в режим фуллскрина (ОР: режим фулскрина, иконка контрола поменялась)
- навести мышкой на первый маркер ( ОР: смена иконки маркера, хинт)
- нажать на маркер (ОР: смена иконки маркера, хинт пропадает)
- навести мышкой на второй маркер (ОР: смена иконки маркера, хинт)
- выйти из режима фуллскрина (ОР: выход из режима фулскрина, иконка контрола поменялась)
- навести на третий маркер (ОР: смена иконки маркера, хинт)

3. (демка: карта с драгабельным маркером у которого есть хинт)
- открыть карту с драгабельным маркером
- навести мышку на маркер (ОР: смена иконки маркера, хинт)
- схватить маркер и перетащить его на на другую улицу (ОР: маркер перенесен на ожидаемую улицу, хинт (при условии, что в тестах мышка будет над маркером)
- нажать на маркер (ОР: сменка иконки маркера, хинта нет)
- зумаут на 10 левел (ОР: тайлы ожидаемой местности 10-го уровня зума)


4. (демка с раскрытыми карочками организаций в балунах. количество балунов = количеству типов графиков работы организаций. каждый тип представляет одна одна организация.
